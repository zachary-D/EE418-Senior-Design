キーボード・アクセラレータ

xfig では、以下のキーボード・アクセラレータ (ショートカット) が使用可能です。

いくつかのポップアップ・パネルの上でも アクセラレータを使用することが可能でとなっています。 詳細については、それぞれのポップアップ・パネルの解説を参照して下さい。

メイン・メニュー

File メニュー

(Meta-F)

下線の付けられた文字は、そのメニューが開かれている時に 使用できるショートカットを示しています。 括弧で囲んで示したショートカット (例えば Meta-X) は、 メニューを開かずに使用することのできるショートカットを示しています。

上の例において、File メニューを開いた状態で 1 を押すと、 ファイル /tmp/sample.fig がロードされます。

アクセラレータ Shift-Meta-P は 以前に設定されたパラメータを用いて印刷を行ないます (つまり、Print パネルは開かれませんが、その図は印刷されます)。

アクセラレータ Shift-Meta-X は 以前に設定されたパラメータを用いてエクスポートを行ないます (つまり、Export パネルは開かれませんが、その図はエクスポートされます)。
